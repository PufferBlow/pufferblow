# Code of Conduct

Pufferblow adopts the [Contributor Covenant, version 2.1][covenant],
unmodified. The canonical text is the source of truth — please read it
there.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This applies in all project spaces — the GitHub organization (issues,
pull requests, discussions, code review comments), our community
servers, and any time someone is representing the project in public.

## Reporting

If you believe someone has acted in a way the covenant prohibits,
contact the maintainers at **`conduct@pufferblow.space`**.

Reports are read only by the maintainer team. Include:

- What happened, when, and where (link if it's online).
- Who was involved (handles or names you observed).
- Any context that helps — links to messages, screenshots, prior
  pattern.

We'll acknowledge within **3 business days** and follow up with the
outcome within **10 business days**. For severe or ongoing harm we
move faster.

## Enforcement

Outcomes follow the covenant's standard enforcement ladder
(correction → warning → temporary ban → permanent ban). Maintainers
have discretion on which step is appropriate for a given incident,
and on whether to disclose the outcome publicly.

For vulnerabilities or security issues, see
[`SECURITY.md`](./SECURITY.md) — those go through a different
channel.
